# Send-Ready Playbook

Use these as structures, not fixed scripts. Replace `〇〇` with one true detail and match her tone.

## Opener

### Distinctive hobby

```text
マッチありがとうございます！
〇〇が好きなんですね。僕も最近気になっていて、最初に始めるなら何がおすすめですか？
```

### Shared interest

```text
〇〇好きなの同じです。
僕は最近△△がよかったんですが、最近のお気に入りありますか？
```

### Tokyo and food

```text
東京でお店を探すの好きなんですね。
僕もカフェ巡りが好きなので、最近行ってよかったところがあれば知りたいです。
```

### Sparse profile

```text
マッチありがとうございます！
休日は外に出る派ですか、それとも家でゆっくり派ですか？
```

Do not use `写真の雰囲気が素敵です` unless no stronger truthful hook exists.

## Ongoing Replies

### Hobby answer

```text
〇〇が一番好きなんですね。
僕は△△派だったんですが、それを聞いたら気になってきました。どんなところが好きですか？
```

### Concrete lab routine

```text
日中は実験でずっと座っていることが多いので、終わった後は体を動かしたくなります。
〇〇さんは普段どんな運動をしますか？
```

### She asks about the user's weekend

```text
休日はドライブかカフェに行くことが多いです。
研究室にいる時間が長い分、休みは外に出たくなります。〇〇さんは最近どこか行きましたか？
```

### Language mismatch

```text
少しでも中国語を話せるのすごいです。
日本語で大丈夫なので、気楽に話しましょう。日本には長く住んでいるんですか？
```

### She shares a difficult experience

```text
それは大変でしたね。
話してくれてありがとうございます。今は少し落ち着きましたか？
```

Do not immediately give advice unless she asks for it.

## Compliments

### She compliments the user

```text
ありがとうございます、そう言ってもらえると嬉しいです。
でも実際はけっこうマイペースです笑
```

### Compliment her taste

```text
その選び方、センスいいですね。
僕もそういう落ち着いた雰囲気の店が好きです。
```

### Interaction-based compliment

```text
〇〇さん、話しやすいですね。
自然にいろいろ聞きたくなります。
```

Use the last one only after a warm exchange.

## Playful Warmth

### Light situational tease

```text
そこまで詳しいなら、もう先生側ですね笑
初心者向けに一つ選ぶなら何にしますか？
```

### Callback

```text
前に言ってた〇〇、ちゃんと覚えてます。
それを聞くと今回も〇〇さんらしいですね笑
```

### Playful future projection

```text
甘いもの好きなら、お店選びは〇〇さんに任せた方がよさそうですね。
僕はコーヒー担当にします。
```

Use future projection only when she has already discussed doing or trying something.

## Low-Energy Replies

### One easier follow-up

```text
そうなんですね。
〇〇は最近始めた感じですか？
```

### One topic pivot

```text
なるほど。
そういえば休日は外に出ることが多いですか？
```

If the next reply is still closed and adds nothing, stop asking.

## Date Bridge

### Cafe/food bridge

```text
そのお店、かなり気になります。
よかったら来週、都内で軽く行ってみませんか？
```

### Conversation-based bridge

```text
話していて楽しいので、もう少しゆっくり話してみたいです。
来週あたり、カフェで軽くお茶しませんか？
```

### She agrees

```text
嬉しいです。
土曜の午後か日曜の夕方だと、どちらが合わせやすいですか？
```

### Area choice

```text
新宿と渋谷ならどちらが行きやすいですか？
行きやすい方でお店を探します。
```

### She is nervous

```text
もちろん急がなくて大丈夫です。
もう少しここで話して、会うなら昼のカフェにしましょう。
```

### She says she is busy without another date

```text
了解です。
落ち着いたタイミングがあれば、また教えてください。
```

Do not immediately offer five more dates.

## LINE

### After date agreement

```text
待ち合わせの連絡もしやすいので、よかったらLINE交換しませんか？
アプリのままでも大丈夫です。
```

### After a good first date

```text
今日はありがとうございました。すごく話しやすくて楽しかったです。
よかったらLINE交換しませんか？
```

### Membership-expired one-shot

```text
はじめまして。〇〇が気になって、メッセージしました。
実は有料会員の期限が切れて、この一通しか送れません。もし少しでも話してみたいと思ってもらえたら、よかったらLINEを追加してもらえると嬉しいです。もちろん無理しなくて大丈夫です。

LINE: 〇〇
```

This exception must state the constraint truthfully. Do not fabricate membership status.

## Repair

### Previous shared topic

```text
そういえば、前に話していた〇〇ってその後どうでしたか？
```

### User replied late

```text
返信遅くなってすみません。
前に話していた〇〇、その後どうでしたか？
```

### Previous message was too strong

```text
少し急に聞きすぎましたね。
無理ないペースで話せたら嬉しいです。
```

Never send a second repair after no response.

## Post-Date

### Interested

```text
今日はありがとうございました。
〇〇の話が特に楽しかったです。また今度、話していた△△にも行きたいです。
```

### Not pursuing

```text
今日はありがとうございました。
お話しできてよかったです。気をつけて帰ってください。
```

Do not promise another date if the user does not want one.

## Suspicious Profile

### Minimal in-app test

```text
マッチありがとうございます。
まずはアプリ内で少しお話しできたら嬉しいです。休日はどんなことをして過ごすことが多いですか？
```

If the reply again pushes LINE, links, money, investment, or side jobs, recommend stopping.

## Profile Structure

Use:

1. identity without status flex;
2. concrete weekday/weekend life;
3. two or three real interests;
4. one playful detail;
5. calm dating intent.

Example:

```text
はじめまして。中国出身で、東京の大学院で研究をしています。

平日は研究室にいることが多いですが、休日はドライブ、ジム、カフェ、映画など、外に出ることが多いです。
車が好きで、車の話になると少しだけ早口になります。

最初は落ち着いていると言われますが、仲良くなるとよく話します。
まずは気軽にお話しして、タイミングが合えばカフェやご飯に行けたら嬉しいです。
```

## Odekake / Date Recruitment

### Cafe or lunch

```text
今週末、都内でカフェか軽くランチに行ける方を探しています。
まずは気軽にお話しできたら嬉しいです。
```

### Driving as a later interest

```text
ドライブが好きです。
まずは気軽にお話しして、仲良くなったら景色のいい場所にも行けたら嬉しいです。
```

Do not recruit a first meeting built around `初対面 + 車 + 夜 + 遠出`.
